比特序列:
    任何一个段文字甚至字母都可以转化为一个byte[], 这过程叫编码, 而这个byte[] 叫做比特序列

XOR: 是针对相同长度的比特序列的一种运算
    异或操作运算规则,参考翻硬币   0: 不翻  1:翻
    0 + 0 = 0 -> (不翻) + (不翻) = (不翻)
    0 + 1 = 1 -> (不翻) + (翻) = (翻)
    1 + 0 = 1 -> (翻) + (不翻) = (翻)
    1 + 1 = 0 -> (翻) + (翻) = (不翻)
